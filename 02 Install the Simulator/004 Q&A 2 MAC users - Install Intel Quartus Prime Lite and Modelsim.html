How to install Modelsim for MAC users?

(thanks to E GH)


The easiest way to solve this is to use the online tool https://edaplayground.com/. You will select from the Tool&Simulators pane Mentor Questa (a better Modelsim) and you can simulate in the browser all the code example from this course.


Intel provides Quartus installation kits only for Windows/Linux so if you want to install Quartus/Modelsim on your Mac you should use some virtual machine with Windows/Linux on it. You should find enough documentation/apps for Mac that successfully do this.


But, as I said, https://edaplayground.com/ should provide enough resources to simulate all the examples from this course.

I hope this helps. Please confirm that you managed to simulate the Verilog "Hello World" example.
