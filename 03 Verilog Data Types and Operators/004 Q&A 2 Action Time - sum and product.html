If you jumped directly to this exercise, please do all the exercises in the suggested order as each exercise helps you gradually building skills using Verilog and Modelsim.

The next steps should solve your problem.


After you created a new project, added the files, compiled them please press Simulation -> Start Simulation.

If you don't see the 'dark page', which is called 'Wave', please click View -> Wave (open_wave.pgn). This will open the Wave panel.


If you don't see the Run simulation option, please Window -> Toolbars -> Simulate (run_simulation.png). This will make visible all the buttons used to run a simulation.
