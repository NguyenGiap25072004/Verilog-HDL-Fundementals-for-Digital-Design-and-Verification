How to monitor values of intermediate variables in the program using test bench?

(thanks to Laxmikant Lokram Barde)


If you want to monitor the value of any variable inside a testbench you can do it in multiple modes.

Here are the easiest 3 ways:


1. Print it automatically in the console whenever that variable changes using the $monitor task inside the testbench:

initial begin
    $monitor(" my_var = %b", my_var);
end


2. Print it in the console during the procedural code execution using the $display task inside the testbench:

initial begin
    sum = a + b;   
    $display(" a = %b, b= %b, sum = %b", a, b, sum);
end


3. Observe it in the wave of the Modelsim simulator when a testbench is simulated.

To do this you need to start a simulation, then click on the instance name of the testbench fom the "sim" pane.

Next you will see all the testbench variables in the "Objects" pane. Select them all and the drag & drop them in the Wave. After this you start the simulation and you will be able to inspect visually how each variable changes in time.


All methods are exercised extensively in our Action Time videos, so you will master them all.
