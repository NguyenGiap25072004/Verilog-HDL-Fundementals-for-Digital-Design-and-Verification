When  write "begin" after each of the if/else statements?

(thanks to Omer Fifer)


The syntax of if else is the following:


if (condition) execute_only_one_statement;

OR

if (condition) begin // for multiple OR only one statement
    execute_statementA1;
    execute_statementA2;
    ...;
    execute_statementAN;
end else begin
    execute_statementB1;
    execute_statementB2;
    ...;
    execute_statementBN;
end
