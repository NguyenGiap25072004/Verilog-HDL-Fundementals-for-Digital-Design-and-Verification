Is timescale needed for every testbench?

(thanks to Minqing Liang)


The timescale directive is used to define exactly what 1 time unit is.

All the simulators have a default value for it, but it is recommended to be defined explicitly in all testbenches, mainly where you create clock signals that have a specific frequency.

At least for this course, you only copy/paste it from one testbench to another so it should be extremely simple to use.
